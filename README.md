# CodeCamp Management System

## 📖 Description
The CodeCamp Management System is a desktop application developed using C# and .NET (Windows Forms). 
This system is designed to manage students, courses, and registrations in a simple and user-friendly way.

## 🎯 Features
- Add, update, and delete student information
- Manage course details
- Register students into courses
- Simple and clean user interface
- Basic data handling and validation

## 🛠️ Technologies Used
- C#
- .NET Framework / .NET
- Windows Forms
- Visual Studio

## 📚 Purpose
This project was developed as part of an academic assignment to demonstrate understanding of:
- Object-Oriented Programming (OOP)
- GUI application development
- Event-driven programming

## 🚀 How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run the application
